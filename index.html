Welcome to PDF Reader

This tool uses AI to make PDF files easier to read and understand. It is made with Python, HTML, CSS, and JavaScript.

You can see the full project here: GitHub Link

What’s the Problem?

  • Reading long PDFs takes a lot of time
  • It’s hard to find the important parts
  • Language can be a barrier
  • Normal PDF apps don’t give summaries

Our AI Solution

  • Uses Gemma AI to read text (even scanned files)
  • Summarizes by full file, by page, or in parts
  • Supports many languages
  • Makes it easier for everyone to understand PDFs

Main Features

Text Extraction

Reads both normal and scanned PDF files

AI Summarizer

Summarizes full text or each page

Search

Helps find keywords fast

Charts

Shows data as charts or tables
